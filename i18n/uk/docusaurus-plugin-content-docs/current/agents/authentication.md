# Автентифікація

Після створення агент знаходиться у статусі **Очікує**. Натисніть **Автентифікувати** на сторінці деталей агента, щоб підключити ваш особистий обліковий запис провайдера. Кожен провайдер має свій потік автентифікації — ви використовуєте ті самі облікові дані, що й для CLI-інструментів провайдера.

## Gemini (OAuth URL)

1. Натисніть **Відкрити URL автентифікації** → увійдіть через ваш особистий Google обліковий запис з підпискою на Gemini
2. Вставте повний JSON-вміст облікових даних із `oauth_creds.json` у текстове поле
3. Натисніть **Надіслати**

Це ті самі облікові дані, які Gemini CLI використовує локально — ваше використання буде зараховуватись до квоти вашої підписки Google/Gemini.

:::tip
JSON перевіряється перед збереженням. Невалідний JSON відхиляється з чітким повідомленням про помилку.
:::

## Claude Code (Ручний токен)

1. Вставте ваш OAuth токен Claude Code з вашого облікового запису Anthropic у текстове поле
2. Натисніть **Надіслати**

Токен зберігається як `agent_token.txt` у контейнері агента. Усе використання працює в межах лімітів вашої підписки Anthropic.

## OpenAI Codex (Device Code)

1. Натисніть **Відкрити URL автентифікації** → завершіть авторизацію пристрою на сайті OpenAI
2. Вставте вміст `auth.json` (або API ключ) у текстове поле
3. Натисніть **Надіслати**

Якщо вставлений вміст є валідним JSON, він зберігається як є. Якщо це API ключ, він автоматично обгортається у `{"api_key": "<ваш-ключ>"}`. Ваш тарифний план OpenAI визначає доступні моделі та ліміти швидкості.

## Повторна автентифікація, відкликання та видалення

| Дія | Що відбувається |
|---|---|
| **Повторна автентифікація** | Замінює існуючі облікові дані новими. Доступно, коли агент вже автентифікований. |
| **Відкликати** | Стирає збережені облікові дані та встановлює статус `revoked`. Агент стає непридатним для використання, але запис залишається. |
| **Видалити** | Повністю видаляє агента та всі пов'язані чат-сесії. |

Усі деструктивні дії потребують підтвердження.

## Безпека облікових даних

- Усі облікові дані **зашифровані** через Active Record encryption
- Максимальний розмір облікових даних: **64 КБ**
- Відкликання агента повністю стирає облікові дані
